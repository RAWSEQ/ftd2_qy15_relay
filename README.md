# ftd2_qy15_relay
ftd2xx relay qy15 controller on linux

# Install
```
$ git clone https://github.com/RAWSEQ/ftd2_qy15_relay.git
$ cd ftd2_qy15_relay
$ make
```

# Usage
```
$ sudo ./ftd2_qy15_relay
Usage: [-port (Port Number)] [-on (Channel)...] [-off (Channel)...]
ex: ./ftd2_qy15_relay -port 0 -on 1 -on 3 -off 2
```
